Le projet HTML

Objectif du projet


Dans le cadre de notre apprentissage, il nous a été donné de créer un site web d'une bibliothèque en ligne. Celui-ci devait être dynamique et disposer de certains standards qui devaient figurer sur le site. Ce projet réalisé durant nos heures d'AP (1 fois par semaine) s'est produit par groupe de 3.

L'objectif principal du projet était de développer un site web d'une bibliothèque totalement fonctionnel et autonome. Tout cela grâce à ce qu'on a appris durant l'année et par l'apprentissage autonome.

Afin d'accéder à cet objectif, nous avons dû respecter quelques contraintes sur le contenu du site. Il devait apparaître sur la première page un widget de la météo par exemple. Pour réaliser le projet, nous avons utilisé principalement du code HTML et CSS pour l'aspect et le contenu du site, mais aussi de PHP afin de rendre celui-ci dynamique. Bien sûr, nous avons dans l'obligation de créer une base de données propre au site afin de gérer les livres ou encore les comptes utilisateurs fictifs. Le lien avec la base de données à été géré via PHP en globalité.

Selon le compte avec lequel on se connecte, on dispose de droits administrateurs dans lesquels on retrouve la gestion des livres en temps réels depuis le site. Ou une utilisation simple de n'importe quel utilisateur.

Le projet était a réaliser par étapes avec un compte-rendu final et le projet final peut être téléchargé ci-dessous.
