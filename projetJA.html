Le projet Java Modération

Objectif du projet


Dans le cadre de l’évolution du site r3st0.fr, une application Java a été développée pour les modérateurs, destinée à être installée sur leurs postes de travail.

Cette application permet de contrôler les avis déposés par les utilisateurs, avec la possibilité de supprimer ou bloquer ceux ne respectant pas la charte d’utilisation du site

L’accès à l’application est sécurisé par un système d’authentification pour chaque modérateur.

Ce projet a été mené en équipe de quatre personnes, selon une approche collaborative

Nous avons travaillé à partir de User Story fonctionnels, avec des réunions régulières pour coordonner l’avancement et ajuster les choix techniques.

L’application repose sur la base de données existante, laquelle a pu être adaptée si nécessaire.

Le projet devait être accompagné de comptes-rendus de chacune des itérations. Vous pouvez les consulter.

Compétences :

Recensement et gestion des ressources numériques : capacité à identifier et organiser les ressources numériques nécessaires à un projet ou une activité. Gestion des demandes : compétence dans la collecte, le suivi et l'orientation des demandes, afin d'assurer une réponse rapide et efficace aux besoins exprimés. Planification des activités : aptitude à structurer et organiser les tâches et les projets pour respecter les délais et optimiser les ressources disponibles.
