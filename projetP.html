Le projet Python

Objectif du projet


Chez STESIO, nous avons été chargés de développer une solution automatisée pour la gestion des bases de données des utilisateurs au sein de l'entreprise. Ce projet était en binôme.

L'objectif principal était de permettre à chaque utilisateur disposant d'un compte dans l'annuaire Active Directory (AD) de bénéficier d'une base de données personnelle, tout en assurant un accès sécurisé et individualisé.

Pour ce faire, il fallait récupérer la liste des utilisateurs en interrogeant l'annuaire AD du contrôleur de domaine CDSTESIO. Ces informations ont ensuite été exportées dans un fichier CSV.

Ensuite, nous avons conçu un script Python capable de lire ce fichier CSV et de générer automatiquement les commandes SQL nécessaires à la création des comptes utilisateurs, des bases de données associées, ainsi que des permissions d'accès. Chaque utilisateur se voit ainsi attribuer une base de données distincte, accessible uniquement par lui-même.

Enfin, le script génère un fichier SQL qui est exécuté sur le serveur de base de données WEB-BD-STESIO. Cette étape leur permet ainsi de se connecter à distance à leur propre espace de travail sur le serveur.

Le projet devait être accompagné de comptes-rendus de chacune des étapes à procéder. Vous pouvez les consulter.
